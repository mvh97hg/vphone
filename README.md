# VPhone

VPhone is a WebRTC SIP softphone for Asterisk/FreeSWITCH and a reusable embedding library for web and Electron applications. The browser global is `Vphone`; use `Vphone.mount()` for new embedded web integrations. Existing `BrowserPhone.mount()` integrations still work as a compatibility alias.

## Runtime Modes

- `Phone/`: standalone phone application. Host this folder when you want the full phone UI.
- `lib/vphone.js`: widget runtime for embedding the phone UI into another website or Electron renderer.
- `lib/vphone.min.js`: single-file production build generated by `npm run build:release` in `lib/`.
- `lib/dist/vphone-sdk.*`: typed headless core facade for custom UI (desktop/browser/mobile).
- `desktop/electron/`: Electron desktop shell that embeds the existing phone UI.

## Quick Start

### Standalone Phone

```bash
cd Phone
python -m http.server 9000
```

Open `http://localhost:9000` and configure SIP settings from the UI, or provision them through embed config.

### Floating Widget

```html
<script src="lib/vphone.js"></script>
<script>
  const phone = Vphone.mount({
    phoneUrl: 'Phone/index.html',
    floating: true,
    width: '250px',
    height: '425px',
    config: {
      wssServer: 'pbx.example.com',
      WebSocketPort: 7443,
      ServerPath: '/ws',
      SipDomain: 'pbx.example.com',
      SipUsername: '1001',
      SipPassword: 'secret',
      Language: 'vi',
      EnableVideoCalling: false,
      MaxDidLength: 24
    }
  });
</script>
```

### Inline Panel

```html
<div id="call-panel" style="width:250px;height:425px"></div>
<script src="lib/vphone.js"></script>
<script>
  Vphone.mount({
    phoneUrl: 'Phone/index.html',
    container: '#call-panel',
    floating: false,
    width: '250px',
    height: '425px',
    config: {
      wssServer: 'pbx.example.com',
      WebSocketPort: 7443,
      ServerPath: '/ws',
      SipDomain: 'pbx.example.com',
      SipUsername: '1001',
      SipPassword: 'secret',
      DisableBuddies: true
    }
  });
</script>
```

### Headless SDK

```js
import { createPhoneClient, createIframeBridgeAdapter } from 'vphone/sdk';
import Vphone from 'vphone/widget';

const widget = Vphone.mount({
  container: '#call-panel',
  floating: false,
  width: '250px',
  height: '425px',
  config: {
    wssServer: 'pbx.example.com',
    WebSocketPort: 7443,
    SipUsername: '1001',
    SipPassword: 'secret'
  }
});

const phone = createPhoneClient({
  sip: {
    wssServer: 'pbx.example.com',
    webSocketPort: 7443,
    serverPath: '/ws',
    sipDomain: 'pbx.example.com',
    username: '1001',
    password: 'secret'
  },
  media: { audio: true, video: false },
  language: 'vi'
}, {
  adapter: createIframeBridgeAdapter(widget)
});

phone.on('registration', event => console.log(event.state));
await phone.register();
await phone.makeCall('1002', { callType: 'audio' });
```

## Media Override (Ringtone/Ringback/Hold/Notification)

Provide media from app config:

```js
Vphone.mount({
  floating: true,
  config: {
    MediaConfig: {
      ringtone: 'Ringtone_1.mp3',
      ringbackTone: 'Tone_EarlyMedia-US.mp3',
      holdMusic: 'speech_orig.mp3',
      notificationSound: 'Alert.mp3'
    }
  }
});
```

Each field accepts:
- file name in `Phone/media`
- absolute URL (`https://...`)
- data URI (`data:audio/...`)

Users can update these values in the in-app `Settings` UI. Saved values override app defaults for that browser/profile.

## Mount Options

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `phoneUrl` | string | `embedded` | Use `Phone/index.html` for external hosting. `embedded` uses the single-file payload when available. |
| `container` | selector/element | `null` | Required for inline mode. |
| `floating` | boolean | `!container` | `true` shows a bottom-right launcher, `false` renders inline. |
| `width` | string | `250px` | Panel width. |
| `height` | string | `425px` | Panel height. |
| `title` | string | `VPhone` | Widget header title. |
| `showStatus` | boolean | `true` | Show online/offline status in widget chrome. |
| `startCollapsed` | boolean | `true` | Floating widget starts collapsed. |
| `expandOnIncomingCall` | boolean | `true` | Auto expand on incoming call events. |
| `config` | object | `{}` | Provisioned phone config passed into `Phone/phone.js`. |
| `onStatusChange` | function | `null` | Receives registration/status events. |
| `onCallEvent` | function | `null` | Receives call events. |

## Common Phone Config

These keys are passed under `mount({ config: { ... } })`. Boolean values can be used directly in embed config.

| Config | Default | Use |
| --- | --- | --- |
| `wssServer` | `null` | SIP WebSocket host. |
| `WebSocketPort` | `null` | SIP WebSocket port. |
| `ServerPath` | `null` | SIP WebSocket path, for example `/ws`. |
| `SipDomain` | `wssServer` | SIP domain/realm. |
| `SipUsername` | `null` | Extension or SIP username. |
| `SipPassword` | `null` | SIP password or HA1 hash. |
| `SipPasswordType` | `plain` | Use `ha1` when passing an HA1 hash. |
| `Language` | `auto` | UI language, for example `vi` or `en`. |
| `MaxDidLength` | `24` | Maximum dial input length. |
| `DidLength` | `6` | Length threshold used for dial/video button state. |
| `IceStunServerJson` | empty | JSON string of STUN/TURN servers. |
| `EnableVideoCalling` | `false` | Enable/disable video call buttons and video answer. |
| `EnableTextMessaging` | `false` | Enable/disable SIP text messaging UI. |
| `EnableTransfer` | `true` | Enable/disable in-call transfer. |
| `EnableConference` | `true` | Enable/disable in-call conference. |
| `DisableFreeDial` | `false` | Hide dialpad/free dial entry. |
| `DisableBuddies` | `false` | Hide manual buddy/contact management. |
| `EnableAlphanumericDial` | `false` | Allow letters in the dial input. |
| `EnableRingtone` | `true` | Play ringtone on inbound calls. |
| `RecordAllCalls` | `false` | Start recording automatically when calls connect. |
| `AutoAnswerPolicy` | `allow` | `allow`, `disabled`, or `enabled`. |
| `DoNotDisturbPolicy` | `allow` | `allow`, `disabled`, or `enabled`. |
| `CallWaitingPolicy` | `allow` | `allow`, `disabled`, or `enabled`. |
| `CallRecordingPolicy` | `allow` | `allow`, `disabled`, or `enabled`. |
| `HideSettingsButton` | `true` in embed | Hide settings entry in embedded mode. |
| `HideRecordAllCallsButton` | `true` in embed | Hide record-all toggle in embedded mode. |

## Keyboard Shortcuts

Available shortcuts in the current UI:

| Context | Shortcut | Action |
| --- | --- | --- |
| Dial input | `Enter` | Start an audio call. |
| Dial input | `Ctrl+Enter` | Start a video call when video calling is enabled. |
| Dial input | `0-9`, `*`, `#`, `+` | Enter dial digits. |
| Dial input | `Backspace` / delete button | Remove the previous digit or selected digits. |
| Chat input | `Enter` | Send the message. |
| Chat input | `Shift+Enter` or `Ctrl+Enter` | Insert a line break. |
| Popup menu | `Escape` | Close the menu. |
| Global | `F2` | Answer the active inbound call. |
| Global | `Escape` | Hang up, cancel, or reject the active inbound call. |
| Global | `F4` | Toggle hold on the active call. |
| Global | `F6` | Toggle mute on the active call. |
| Global | `F8` | Toggle transfer mode on the active call. |
| Global | `F9` | Open the dialpad tab. |

Use `KeyboardShortcuts` in phone config to remap global shortcuts when a browser or operating system shortcut conflicts. Example: `KeyboardShortcuts: { answer: "F3", hangup: "Escape", hold: "F5", mute: "F7", transfer: "F10", dialpad: "F12" }`.

## Reconnect Behavior

The phone auto-reconnects the SIP WebSocket when the transport fails and when the browser or Electron renderer returns online. Configure the retry loop through:

| Config | Default | Use |
| --- | --- | --- |
| `TransportReconnectionAttempts` | `999` | Maximum reconnect attempts after transport loss. |
| `TransportReconnectionTimeout` | `3` | Seconds between reconnect attempts. |

Example with feature toggles:

```js
Vphone.mount({
  phoneUrl: 'Phone/index.html',
  floating: true,
  config: {
    EnableVideoCalling: true,
    EnableTransfer: false,
    EnableConference: false,
    EnableTextMessaging: false,
    RecordAllCalls: false,
    CallRecordingPolicy: 'disabled',
    AutoAnswerPolicy: 'disabled',
    DoNotDisturbPolicy: 'allow',
    DisableBuddies: true,
    MaxDidLength: 24
  }
});
```

## WebRTC Media Path

The library uses SIP.js and `RTCPeerConnection`. SIP signaling goes through the SIP WebSocket server. Media can be direct peer-to-peer, relayed through TURN, or anchored by the PBX/SBC depending on SDP/ICE and server configuration. Configure STUN/TURN with `IceStunServerJson` when clients are behind NAT.

```js
IceStunServerJson: JSON.stringify([
  { urls: 'stun:stun.l.google.com:19302' },
  { urls: 'turn:turn.example.com:3478', username: 'user', credential: 'pass' }
])
```

## Build And Test

```bash
cd lib
npm test
npm run build:release
```

`build:release` rebuilds the SDK artifacts and `lib/vphone.min.js`.

## Electron Desktop

```bash
cd desktop/electron
npm install
npm start
```

By default the desktop app opens without SIP credentials so the in-panel Settings menu can be used first. Place `config.json` next to the executable, or set `VPHONE_DESKTOP_CONFIG`, to pre-provision SIP settings. If pnpm blocks Electron build scripts, run `pnpm approve-builds` and allow `electron`, then run `pnpm install` again. See `docs/electron/README.md` for package/build details.
